# Codex-CLI v0.142.0 斜杠指令极简速查表
## 总说明
1. 适用版本：`@openai/codex 0.142.0`，网页端Codex、本地CLI通用
2. 执行规则：指令以 `/` 开头，单独一行输入优先级最高；支持单指令/组合指令串联
3. 资源消耗：`/model max`、`/agent` 高消耗Token，批量任务建议分批执行
4. 终止指令：任意长任务执行异常时输入 `/stop` 强制中断

## 一、会话状态与上下文管控指令
| 指令 | 核心作用 | 最简使用案例 | 适用场景 | 注意事项 |
|------|----------|--------------|----------|----------|
| `/status` | 查看当前会话完整信息：模型、权限、沙箱、Token余量、文件读写范围 | `/status` | 排错前校验环境、确认额度、查看权限 | 无副作用，复杂工程必前置执行 |
| `/compact` | 压缩冗余对话上下文，保留代码/规则，释放上下文窗口 | `/compact` | 多轮迭代后输出混乱、上下文溢出 | 不会删除已有代码、固化技能规则 |
| `/clear` | 彻底清空全部会话历史，重置会话环境 | `/clear` | 新项目切换、历史对话严重污染 | 所有历史对话全部清除，无法恢复 |
| `/log` | 导出当前会话完整操作日志、修改记录、报错堆栈 | `/log` | 追溯文件变更、定位隐性BUG、提交反馈 | 日志包含完整文件修改内容，敏感项目脱敏后导出 |
| `/feedback` | 唤起官方反馈面板，自动携带当前会话日志 | `/feedback` | 上报功能异常、逻辑缺陷、需求建议 | 涉及项目代码建议先脱敏隐私信息 |

## 二、云端沙箱环境指令
| 指令 | 核心作用 | 最简使用案例 | 适用场景 | 注意事项 |
|------|----------|--------------|----------|----------|
| `/cloud` | 切换至云端隔离沙箱模式，所有文件远程运行 | `/cloud` | 本地环境缺失、依赖冲突、纯净测试 | 会话关闭后沙箱文件自动销毁，重要文件提前导出 |

## 三、/model 模型调度指令（三档位推理控制）
| 指令 | 核心作用 | 最简使用案例 | 适用场景 | 注意事项 |
|------|----------|--------------|----------|----------|
| `/model` | 调出模型档位选择面板，切换推理策略 | `/model` | 按需切换速度/精度模式 | 仅当前会话生效，新建会话重置均衡模式 |
| `/model speed` | 极速档位：低算力、响应最快 | `/model speed` | 代码格式化、批量注释、简单语法纠错 | 复杂架构易出现逻辑漏洞，不用于重构/算法开发 |
| `/model balance` | 均衡档位（默认）：速度&精度平衡 | `/model balance` | 日常功能开发、小型项目迭代 | 绝大多数常规开发首选 |
| `/model max` | 深度思考档位：多轮自检、强严谨性、高Token消耗 | `/model max` | 大型重构、架构设计、深度漏洞排查、复杂算法 | 高频调用易触发速率限制，大型项目分模块执行 |

## 四、/agent 自主智能体自动化指令
| 指令 | 核心作用 | 最简使用案例 | 适用场景 | 注意事项 |
|------|----------|--------------|----------|----------|
| `/agent` | 激活Agent自主循环，开启任务自动拆解、分步执行 | `/agent` | 全项目批量改造、无人值守流水线 | 开启前备份项目，默认只读权限，需手动放开编辑 |
| `/goal [任务描述]` | 定义Agent终极目标，全自动闭环执行 | `/goal 重构src全部代码，统一规范并输出变更报告` | 长周期工程自动化、一站式项目优化 | 目标描述必须清晰，禁止模糊需求 |
| `/stop` | 强制终止Agent循环、中断模型推理、停止文件修改 | `/stop` | Agent执行出错、修改不符合预期、额度不足 | 中断后已修改文件不会自动回滚，需手动恢复 |

### Agent权限控制补充（配套参数）
1. 只读模式：`/agent read-only` — 仅扫描分析，不修改文件（新手推荐）
2. 编辑模式：`/agent edit` — 允许修改代码文件（默认常用）
3. 终端模式：`/agent terminal` — 允许自主执行shell/npm指令（谨慎使用）

# 工业级标准组合指令模板（直接复制）
## 模板1：大型项目深度重构流程
```
/status
/compact
/model max
/agent edit
/goal 完整重构当前项目，统一编码规范、修复全部漏洞、优化性能，输出完整变更日志
```
## 模板2：轻量批量格式化（低消耗）
```
/status
/model speed
/skills run 代码格式化
```
## 模板3：安全代码审查（只读不修改）
```
/clear
/cloud
/agent read-only
/goal 扫描项目所有代码，输出风险审查报告，不修改任何文件
```
